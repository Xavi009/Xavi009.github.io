How It's Made

I'm really proud of this project because I made this during my sophomore year in high school when I was just getting into 3D printing and 3D modelling, so I really didn't know what I was doing but I stuck through it. I started by importing a T-Pose SSBM Captain Falcon model from The Models Resource into Autodesk 3ds Max 2016 and modified it using a biped skeleton. This allows me to map an existing mesh onto a human skeleton, allowing me to shape the existing model into a dynamic pose. Finally, I repaired the exported .STL file from 3ds Max into Microsoft's STL Repair website. This website analyzes and repairs any open surfaces in the .STL file and optimizes the model for 3D printing. The final result was better than anything I would have hoped for.

Download the files on my Thingiverse website.
