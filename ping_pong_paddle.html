How It's Made

This was my first original design that was iteratively developed using my own 3D printer back in junior year of high school. I learned how to design parts that are feasible to manufacture and assemble using an FDM 3D printer. The inspiration for this was to bring my passion for tennis into the table tennis world.

I designed the part using the then free-to-use Fusion 360. I started by importing side images of a ping pong paddle and creating sketch outlines of the general shape of a regulation paddle. I then modified the paddle to operate like a tennis racquet, paying special attention to having an even number of cross and main strings. I then had my buddy string the ping pong paddle for me using actual tennis strings. This completely warped the plastic since the stringing machine applies too much tension. The strings were replaced with floss.

Download the files on my Thingiverse website.

Now I bet you're wondering if this worked...

No...it's incredibly harder to play ping pong with this. With this design it's possible to "frame" a shot since there is a frame now. Additionally, the lack of friction between string and ball prevents a player from applying spin or slice effects.

All in all, I'm super happy with how my first true 3D model turned out. It was a fun joke to make.
